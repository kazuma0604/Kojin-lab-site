# 個人開発Lab

LP・WordPress制作 / 個人開発の受託サイト（ポートフォリオ兼ランディングページ）。

🔗 **公開URL**: https://kazuma0604.github.io/kojin-lab-site/

## 特徴

- ダーク＋グローのモダンなデザイン（Vercel/Linear系）
- パーティクルネットワークの動く背景（Canvas）
- スクロール連動アニメーション・現在地ハイライト
- フローティングお問い合わせボタン
- 完全レスポンシブ（モバイル/タブレット/デスクトップ）
- ビルド不要 — `index.html` を開くだけで動作

## 構成

- `index.html` — マークアップ
- `style.css` — スタイル（CSS変数ベース、ダークテーマ）
- `main.js` — アニメーション・ナビ・パーティクル背景（依存ライブラリなし）

## 技術

Vanilla HTML / CSS / JavaScript。フレームワーク・ビルドツールなし。
このサイト自体が「軽量・高速・依存最小」の制作方針のサンプルです。
